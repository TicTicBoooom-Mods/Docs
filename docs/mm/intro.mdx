The config file can be located here,<br/>
`(modpackName)\config\masterful_machinery\controllers\(controllerName.json)`

Recipes can be loaded with either `Vanilla datapack`, `openloader`, `kubeJS` or `craftTweaker`

Vanilla datapack recipes - `(datapackName)\data\mm\recipes\(recipeName.json)`<br/>
Vanilla datapack structures - `(datapackName)\data\mm\recipes\structures\(structureName.json)`<br/>
Openloader recipes - `(modpackName)\openloader\data\(datapackName)\data\mm\recipes\(recipeName.json)`<br/>
Openloader structures - `(modpackName)\openloader\data\(datapackName)\data\mm\recipes\structures\(structureName.json)`<br/>
KubeJS recipes - `(modpackName)\kubejs\data\mm\recipes\(recipeName.json)`<br/>
KubeJS structures - `(modpackName)\kubejs\data\mm\recipes\structures\(structureName.json)`<br/>
CraftTweaker recipes - `(modpackName)\scripts\mm\recipes.zs`<br/>
CraftTweaker structures - `(modpackName)\scripts\mm\structures.zs`
